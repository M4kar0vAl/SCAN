# Проект СКАН

## Установка зависимостей

`npm install`

## Переменные окружения

- Создать файл `.env` в корне проекта
- Указать в нем все необходимые переменные (см. `example.env`)

## Запуск

`npm run dev`

## В докере

### Запуск

В корне проекта:

   - `docker build -t scan .`
   - `docker run --rm -d -p 5173:5173 --name scan scan`

### Остановка

`docker stop scan`
